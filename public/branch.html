Branch Data

Get Addresses of Branch Locations That Has Bought a Machine in Past __ Month



Get Total Cost of All Machines for Each Branch


Get View that Shows Number of Customers Per Branch



SQL query statement

  • Branch Data: 'SELECT * FROM Branch'
  • Get Addresses: 'SELECT b.address FROM Branch b, Machine m WHERE m.branch_id = b.branch_id and m.date_bought >= date_trunc('month', current_date- interval '" + month + "' month)'
  • Get Total Cost By Branch: 'Select sum(cost) as Total_Cost , branch_id from Machine group by branch_id'
  • Create View: 'Create or Replace view custBranch as Select count(email_address) as CustomersInBranch, branch_id From GoesTo group by branch_id '
  • Displaying View: 'SELECT * FROM custBranch '
